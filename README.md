# 用户打赏支持二维码插件文档

## 项目介绍
### 1. 注册了函数工具可以让bot在收到用户打赏请客支持的意图时，发送打赏二维码
### 作者：雷诺哈特
例如：
- 1.当用户表达打赏、请客、支持等意图时，可以调用send_support_image工具发送打赏二维码。
- 2.可以在人格中增加相关提示，让bot在合适的时机主动发送打赏二维码。



## 核心功能
### 1. 发送打赏二维码
- 当LLM收到用户打赏请客支持的意图时，发送打赏二维码
- 支持自定义二维码本地路径（通过Web面板或配置文件）
- 发送二维码时记录详细日志

### 2. 配置管理
- 支持通过Web面板或配置文件设置二维码本地路径
- 默认使用插件目录下的support_image.png文件

## 配置说明
### 可配置参数
| 配置项名称 | 说明 | 默认值 |
|-----------------------------|---------------------------------------|---------|
| `support_image_path` | 用户打赏支持时发送的打赏二维码本地路径 | support_image.png |
| `support_thank_text` | 用户打赏支持时发送的感谢文本 | 已成功发送打赏二维码，十分感谢！ |

### 配置规则
- 初始配置可通过Web面板或配置文件设置
- 修改配置后需重启插件生效
- 二维码文件需放置在插件目录或指定的路径下

## **使用说明**
- 在Bot的人格设置中，添加相关提示，让Bot在收到用户打赏、请客、支持等意图时，调用`send_support_image`工具
- 例如："当用户表达打赏、请客、支持等意图时，请调用send_support_image工具发送打赏二维码"
- 将打赏二维码文件放置在插件目录下，命名为support_image.png（或通过配置指定其他路径）

## **注意事项**
- 二维码文件需要存在于指定的本地路径
- 建议使用尺寸适中的图片，确保发送和加载速度
- 发送二维码操作会记录详细日志，便于追溯
